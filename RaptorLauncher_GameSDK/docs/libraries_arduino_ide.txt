Bibliotheques Arduino IDE a installer
====================================

SDK de base (obligatoire):
- LovyanGFX
- XPT2046_Touchscreen
- Adafruit MCP23017 (Adafruit_MCP23X17)
- ArduinoJson
- ESP32 board package (Espressif)

Incluses avec ESP32 core (pas d'installation séparée):
- Preferences
- esp_ota_ops / esp_partition
- WiFi

Fonctions avancées (optionnelles):
- PNGdec (PNG custom)
- JPEGDEC (JPEG custom)
- ESP8266Audio (WAV/MP3)
